# Santa Point App

Santa Point App は、3〜7歳の子どもがサンタさんに日々の「できた！」を報告してポイントを貯め、ランクアップに応じて担当サンタやお部屋の見た目が変わっていく Expo / React Native アプリです。

## このアプリで実現したいこと

Santa Point App で実現したいのは、子どもが毎日の小さな「できた！」を、やらされ感なく楽しく積み重ねられる体験です。

サンタさんに話しかけることで、努力や成長がやさしく肯定され、続けること自体がうれしくなる設計を目指しています。親にとっては、しつけや評価のための管理ツールではなく、子どもの前向きな行動を一緒に喜べる補助線のような存在にしたいと考えています。

## 解決したい課題

このサービスは、以下のような日常の課題を解決する手法として設計しています。

- 子どもの小さながんばりは、その場で流れてしまいやすく、継続の実感につながりにくい
- 親が「できていないこと」を注意する会話が増えると、親子ともに負担になりやすい
- 習慣化アプリは、子どもにとって「やらされるもの」「管理されるもの」になりやすい
- ごほうび設計だけに寄ると、行動の楽しさより報酬が主役になってしまう
- 毎日の成長を、親子で前向きに振り返るきっかけが少ない

Santa Point App は、これらの課題に対して「サンタさんに話しかける」というやさしい体験を入口にしながら、子どもの行動を前向きに積み上げられる形を目指しています。

## コアバリュー

このサービスのコアバリューは、子どもの毎日の小さな前進を、評価ではなく“愛着のある体験”として積み上げられることです。

特に重視している価値は以下です。

- 小さな「できた！」が肯定されること
- 習慣がプレッシャーではなく楽しみとして続くこと
- 子どもがサンタさんとの関係に愛着を持てること
- 親が管理者ではなく、見守る立場で関われること
- 継続によって、ポイントだけでなく気持ちの積み重なりが残ること

## 感情設計

Santa Point App は、単にポイントを貯める習慣化アプリではなく、「子どもの前向きな行動がやさしく肯定される体験」を中心に設計しています。

### 目指す感情の流れ

1. 話しかけたくなる
2. ほめられてうれしい
3. またやってみたくなる
4. 少しずつ自信になる

### 子どもに対する感情設計

- 評価される緊張感より、話したくなる親しみやすさを優先する
- ミッション達成感より、「できたことを聞いてもらえるうれしさ」を重視する
- ごほうび依存ではなく、ほめられる体験そのものが楽しみになることを目指す
- サンタさんが成長と一緒に変化することで、継続が物語のように感じられるようにする

### 親に対する感情設計

- 管理や監視の道具ではなく、見守りの補助線として感じられること
- 子どもを責めるきっかけではなく、良かった点に目を向けるきっかけになること
- 毎日の小さな報告を通して、親子の会話が自然に増えること

### デザインと体験への反映

- ダークで落ち着いた背景に、赤や金のあたたかさを差し込み、安心感と魔法感を両立する
- サンタのお部屋やランクアップ演出で、成長が視覚的に報われる感覚をつくる
- おてがみや会話文面は、評価口調ではなく、やさしく受け止めるトーンを基本にする

## 概要

- 音声またはテキストでサンタさんに報告できる
- 報告ごとに 1〜3pt をランダム付与
- 累計ポイントでランクアップし、担当サンタが変化する
- お部屋ビジュアルがランクや時間帯に応じて変化する
- ウィッシュリストとおてがみ機能がある
- 複数プロフィールをローカル保存できる

## 主要仕様

### ポイントとランク

- 報告ごとに `1〜3pt` をランダム付与
- `pointsThisYear`
  - 毎年 1/1 にリセット
  - プレゼント交換など年間利用向け
- `pointsAllTime`
  - リセットなし
  - ランク計算に使用

ランク閾値:

| ランク | 累計ポイント |
| --- | ---: |
| 1 | 20 |
| 2 | 60 |
| 3 | 120 |
| 4 | 220 |
| 5 | 360 |
| 6 | 560 |
| 7 | 840 |
| 8 | 1220 |
| 9 | 1750 |
| 10 | 2500 |

### 担当サンタ同期

現在ランクのサンタが、以下に一貫して反映されます。

- ホーム画面の部屋 / サンタ表示
- 会話モーダル
- ランクアップ演出
- おてがみ文面
- 会話返答テキスト

### おてがみ

- 初回作成時
  - `はじめまして！` の 1 通だけを生成
- 月次おてがみ
  - 前月ぶんの報告回数
  - 前月にもらったポイント合計
  - 前月によく出た話題
  をもとにテンプレート生成
- 生成タイミング
  - 起動時やプロフィール切り替え時など、プロフィール正規化タイミング

### 誕生日入力

- 初回オンボーディング
- 子ども情報編集モーダル

の両方で、数字入力に応じて `YYYY/MM/DD` に自動整形されます。保存時は `YYYY-MM-DD` に正規化します。

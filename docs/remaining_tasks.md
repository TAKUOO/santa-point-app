- 初回リリース時に用意する room 素材は **ランク4（ゴールドサンタ）まで**

  - 4ランク x 4時間帯 = 16パターン
- ランク5以降（プラチナ〜にじいろ）は翌月以降に段階追加
  - ランク5〜7: 12パターン
